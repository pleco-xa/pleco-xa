Oscillator — pleco vs native Web Audio

pleco-xa/engine · OscillatorNode

This is the whole thesis of pleco's audio engine on one page. The same graph — an OscillatorNode into the destination — is built twice: once on pleco's zero-dependency PlecoOfflineAudioContext, once on the browser's own native OfflineAudioContext. Both render offline, and we compare the two buffers sample for sample. For a pure sine the outputs are identical; for the band-limited square and sawtooth they match to a hair, and where they don't it is because the spec deliberately leaves the anti-aliasing curve to the user agent — an honest, documented divergence, not a bug. The Play button plays pleco's rendered samples through the browser (pleco synthesizes; the browser is just the speaker). Same asserts run headless in examples/node/engine-oscillator.mjs.
