server, no Python, no pre-baked results. Each page loads the same code you'd
`npm install`, points it at real audio, and shows the analysis live. Open any
of them, then hit "view source" on the demo to see the exact calls.

## Run the demos locally

The gallery mirrors the standalone example pages in
[`examples/web/`](https://github.com/brookcs3/pleco-xa/tree/main/examples/web).
Serve the repository root over HTTP and open any file there directly, or run
the docs site — the `predev`/`prebuild` hooks copy the demos, the built bundle,
and the audio corpus into place automatically.
