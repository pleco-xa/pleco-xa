This demo pulls the *Orphans* mix, vocals, and instrumental stems and runs two
different separators over the full mix. The first is the REPET-SIM soft-mask
recipe (STFT, similarity mask with margins 2/10 and power 2, inverse STFT). The
second is Pleco-Xa's own fingerprint-and-reconstruct path. Because the true
stems are on hand, you can hear and measure how close each estimate lands to the
real vocal and accompaniment.

<DemoEmbed src="plot-vocal-separation.html" title="REPET-SIM vs fingerprint vocal separation" height={800} />

The masking path is built from `stft`, `istft`, and the `decompose` namespace's
`softmask`; the fingerprint path uses `decompose.processAudioToFingerprints` and
`decompose.reconstructVocal`. Audio comes in through `decodeWav`, so the whole
pipeline is plain arrays in, plain arrays out.
