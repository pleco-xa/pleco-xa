Structural segmentation is where several of Pleco-Xa's primitives compose into
one algorithm. This demo builds a beat-synchronous, z-scored feature stack (12
chroma over 13 MFCC), constructs a recurrence affinity matrix, takes the
eigenvectors of its normalized graph Laplacian, and clusters them with k-means —
the actual McFee–Ellis method, not an approximation. With `k = 3`, the returning
chorus earns a single label that lights up as an off-diagonal block in the
affinity matrix.

<DemoEmbed src="laplacian-segmentation.html" title="McFee–Ellis Laplacian segmentation" height={820} />

The headline call is `segment.laplacianSegmentation`, fed by
`segment.recurrenceMatrix`, `beat_track`, the `feature` namespace, and `sync`
for the beat-synchronous aggregation. A synthetic control case in the same page
recovers planted boundaries exactly, so the real-audio result isn't just a
pretty picture.
