Pleco-Xa is a browser-native audio analysis engine. It brings musical
intelligence — BPM and beat tracking, spectral features, structural
segmentation, and its signature loop detection — to any environment that runs
JavaScript, with no build step and no dependencies.

It also, you'll discover, covers everything librosa does — validated function by
function against it — while adding real-time and loop-analysis capabilities that
an offline Python library structurally cannot.
