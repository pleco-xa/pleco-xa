Pleco-Xa is a browser-native audio analysis engine. It brings musical
intelligence — BPM and beat tracking, spectral features, structural
segmentation, effects, pitch tracking, and its signature loop detection — to any
environment that runs JavaScript, with no build step and no dependencies.

Built for music. Runs anywhere. And it does things an offline library can't —
real-time streaming analysis and intelligent loop-point detection. Every example
in the [gallery](/gallery/) runs live in your browser, on your own audio.
